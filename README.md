# tod-ReID
淘然视界行人ReID数据集（如有问题，可以随时在Issues中提问）：<br>
1）数据规模：当前165人，持续更新<br>
2）数据种类：图片+点标注<br>
3）采集设备：IPC<br>
4）种族分布：黄种<br>
5）采集场景：室外<br>
6）年龄分布：未知<br>
7）角度姿态：多角度多姿态<br>
8）人脸表情：中立、闭眼<br>
9）光照分布：未知<br>
10）分辨率：478 * 768<br>
11）数据格式：jpg、png、json<br>
12）标注内容：12点<br>
